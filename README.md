# lbd-receipt-scraper

Learn by doing - 抓下台灣統一發票對獎號碼

## Usage

```bash
pip install -r requirements.txt
python app.py
```

## Result

```
01099147
55517602
16246076、78369053、06816904
021
```

## Dependencies

- [beautifulsoup4](http://www.crummy.com/software/BeautifulSoup/bs4/)
Screen-scraping library
